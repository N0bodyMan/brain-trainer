Цель: удержать четырёхзначное число в памяти и мысленно прибавить к каждой цифре заданное число — не глядя на исходное.

  1. Цифры числа мелькают по одной — запомни их
  2. Число скрывается — появляется задание, например +2
  3. Прибавь 2 к каждой цифре в уме и введи результат
  4. Ответил — задание исчезает, появляется следующее (+3, +4…)
  5. После всех 5 заданий — итоговый результат

Счёт по модулю 10: цифры только 0–9, после 9 идёт 0. Например: 7+4=1, 9+2=1.

👁 рядом с полем — подсмотреть ответ. Число всегда можно раскрыть на карточке сверху.

Каждые 2.5 секунды появляется новая цифра. Нажми «Совпадает!» если текущая цифра совпадает с той, что была ровно N шагов назад.

  1. Первые N цифр — просто запоминай, кнопки заблокированы
  2. Когда появляется новая цифра — сравни её с той, что была N шагов назад
  3. Совпадает → жми «Совпадает!»; не совпадает → ничего не делай (или «Нет»)

Пример (2-back): 7 → 3 → 7 ← совпадает с шагом 1! → 5 → 3 ← совпадает с шагом 2!

Тренирует: рабочую память, способность одновременно удерживать и обновлять информацию.
